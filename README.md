A native macOS app for culling a large Photos library via keyboard. Built with Swift, SwiftUI, and Apple's Photos framework.

You must install Xcode: https://developer.apple.com/xcode/

Open the project in Xcode by running open PhotoCuller.xcodeproj.
The app requires macOS 13 (Ventura) or later and Xcode 15+. On first launch, you will be prompted to grant Full Access to your Photos library.
